创建一个完整的单文件PWA移动应用「五月天演唱会小本本」，文件名为 index.html。

## 功能需求

### 1. 首页 - 倒计时
- 顶部大字显示距离下一场五月天演唱会的倒计时（天/时/分/秒）
- 如果没有设置下一场，显示「暂无下一场信息」，点击可设置
- 抢票日提醒条（如果有设置）
- 底部快捷入口：足迹地图、打卡记录、分享卡片

### 2. 足迹地图
- 中国地图简化版（用SVG画省份轮廓或用Canvas绘制）
- 去过的城市在地图上高亮标记（蓝色圆点+城市名）
- 下方列表展示每场详情：城市、场馆、日期、座位区
- 预填7场：北京/青岛/上海/太原/杭州/香港/深圳

### 3. 打卡记录
- 表单：城市选择器、场馆（文本输入）、日期选择器、座位区（文本）、本场歌单（textarea）
- 保存到 localStorage
- 支持编辑和删除
- 打卡后自动更新足迹地图

### 4. 分享卡片
- Canvas 渲染生成图片卡片
- 卡片内容：五月天风格标题 + 已去过城市名称列表 + 场次总数 + 「X城X场五迷认证」
- 一句随机五月天歌词在卡片底部
- 点击保存图片到手机

### 5. 设置
- 设置下一场演唱会日期和城市
- 设置抢票日期

## 设计规范

- 五月天主题色：#1a5fdc（蓝色）为主色，#ff6b35（橙色）点缀
- 背景深色系 #0f1923，卡片深蓝 #1a2a3a
- 字体：PingFang SC / Microsoft YaHei
- 移动端优先，max-width 500px
- 底部导航栏：首页 | 足迹 | 打卡 | 分享
- 卡片圆角 12px，微阴影
- 图标用 emoji

## 技术实现

- 纯单文件 HTML，CSS + JS 全部内嵌
- localStorage 存储数据
- PWA 完整支持：manifest.json（内联data URI）、service worker 支持离线缓存
- 响应式适配，viewport 设置
- 分享卡片用 Canvas API 绘制
- 倒计时每秒刷新
- 内置5首随机五月天歌词（倔强、突然好想你、温柔、恋爱ing、知足的经典歌词）

## 预填数据

如果 localStorage 为空，预填以下7个场次记录（城市名填好，场馆和日期留空）：
北京、青岛、上海、太原、杭州、香港、深圳

## 微信兼容

- data URI 中的中文使用 encodeURIComponent
- 分享功能提供 textarea fallback
- apple-mobile-web-app 相关 meta 标签
